fun addListener(aListener: KSetListener<E>): Unit

Adds an KSetListener which will be notified whenever elements are added to or removed from the KObservableSet. If the same listener is added more than once, then it will be notified more than once.

Parameters

aListener - the KSetListener to be registered.
