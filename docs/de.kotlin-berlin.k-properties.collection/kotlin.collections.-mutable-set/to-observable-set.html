toObservableSet

fun <E> MutableSet<E>.toObservableSet(): KObservableSet<E>

Creates a new KObservableSet instance that is backed by this MutableSet.

Note that any changes made directly to this MutableSet will not be observed. Only the returned KObservableSet should be used to make changes.
