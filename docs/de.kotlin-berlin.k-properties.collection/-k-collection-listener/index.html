KCollectionListener

interface KCollectionListener<E>

An KCollectionListener is notified whenever elements are added to or removed from an KObservableCollection. It can be registered and unregistered with KObservableCollection.addListener respectively KObservableCollection.removeListener. The same instance of KCollectionListener can be registered to listen to multiple KObservableCollection instances.
