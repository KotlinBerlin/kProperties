KObservableMap

interface KObservableMap<K, V> : MutableMap<K, V>

A Map that can be observed for changes. The KObservableMap supports observation of added, removed and replaced key value mappings through the KMapListener interface.

KMapListener instances can be added and removed with the addListener and removeListener methods.
