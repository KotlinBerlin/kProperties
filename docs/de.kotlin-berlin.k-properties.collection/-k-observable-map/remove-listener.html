removeListener

abstract fun removeListener(aListener: KMapListener<K, V>): Unit

Removes the given listener from the list of listeners, that are notified whenever elements are added to, removed from or replaced in the KObservableMap.

If it had been previously added then it will be removed. If it had been added more than once, then only the first occurrence will be removed.

Parameters

aListener - the KMapListener to be removed.
