KObservableList

interface KObservableList<E> : KObservableCollection<E>, MutableList<E>

A List that can be observed for changes. The KObservableList supports observation of added and removed elements as well as replacements and movements of elements through the KListListener interface.

KListListener instances can be added and removed with the addListener and removeListener methods.

Functions

addListener

Adds an KListListener which will be notified whenever elements are added to, removed from, replaced or moved in the KObservableList. If the same listener is added more than once, then it will be notified more than once.

abstract fun addListener(aListener: KListListener<E>): Unit

removeListener

Removes the given listener from the list of listeners, that are notified whenever elements are added to, removed from, replaced or moved in the KObservableList.

abstract fun removeListener(aListener: KListListener<E>): Unit

When using the implementations of this library, then he returned list is currently not observable and doesnt trigger any change events on the main list if any modifications are made through the sub list. This may change in the future.
