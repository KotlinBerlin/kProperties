fun addListener(aListener: KListListener<E>): Unit

Adds an KListListener which will be notified whenever elements are added to, removed from, replaced or moved in the KObservableList. If the same listener is added more than once, then it will be notified more than once.

Parameters

aListener - the KListListener to be registered.
