WeakKCollectionListener

class WeakKCollectionListener<E> : KCollectionListener<E>

A WeakKCollectionListener can be used if an KObservableCollection should only maintain a weak reference to the listener. This helps to avoid memory leaks that can occur if observers are not unregistered from observed objects after use.
