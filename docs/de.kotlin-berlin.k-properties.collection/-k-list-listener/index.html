KListListener

interface KListListener<E>

An KListListener is notified whenever elements are added to, removed from, replaced or moved in an KObservableList. It can be registered and unregistered with KObservableList.addListener respectively KObservableList.removeListener. The same instance of KListListener can be registered to listen to multiple KObservableList instances.
