toObservableMap

fun <K, V> MutableMap<K, V>.toObservableMap(): KObservableMap<K, V>

Creates a new KObservableMap instance that is backed by this MutableMap.

Note that any changes made directly to this MutableMap will not be observed. Only the returned KObservableMap should be used to make changes.
