removeListener

abstract fun removeListener(aListener: KCollectionListener<E>): Unit

Removes the given listener from the list of listeners, that are notified whenever elements are added to or removed from the KObservableCollection.

If it had been previously added then it will be removed. If it had been added more than once, then only the first occurrence will be removed.

Parameters

aListener - the KCollectionListener to be removed.
