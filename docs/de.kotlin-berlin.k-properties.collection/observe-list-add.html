observeListAdd

inline fun <E> KObservableList<E>.observeListAdd(crossinline aCallback: (KObservableList<E>, Collection<E>) -> Unit): KListListener<E>

Creates a new KListListener instance, that calls aCallback whenever the KListListener.onAdd method is called. The new listener is automatically added to this KObservableList instance and then returned.
