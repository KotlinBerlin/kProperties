observeMapAdd

inline fun <K, V> KObservableMap<K, V>.observeMapAdd(crossinline aCallback: (KObservableMap<K, V>, Collection<Pair<K, V>>) -> Unit): KMapListener<K, V>

Creates a new KMapListener instance, that calls aCallback whenever the KMapListener.onAdd method is called. The new listener is automatically added to this KObservableMap instance and then returned.
