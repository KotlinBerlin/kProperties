observeSetAdd

inline fun <E> KObservableSet<E>.observeSetAdd(crossinline aCallback: (KObservableSet<E>, Collection<E>) -> Unit): KSetListener<E>

Creates a new KSetListener instance, that calls aCallback whenever the KSetListener.onAdd method is called. The new listener is automatically added to this KObservableSet instance and then returned.
