removeListener

open fun removeListener(aListener: KListListener<E>): Unit

Removes the given listener from the list of listeners, that are notified whenever elements are added to, removed from, replaced or moved in the KObservableList.

If it had been previously added then it will be removed. If it had been added more than once, then only the first occurrence will be removed.

Parameters

aListener - the KListListener to be removed.
