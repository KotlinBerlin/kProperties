removeListener

abstract fun removeListener(aListener: KChangeListener<T>): Unit

Removes the given listener from the list of listeners, that are notified whenever the value of the KObservableValue changes.

If it had been previously added then it will be removed. If it had been added more than once, then only the first occurrence will be removed.

Parameters

aListener - the KChangeListener to be removed.
