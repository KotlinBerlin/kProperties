addListener

open fun addListener(aListener: KCollectionListener<E>): Unit

Adds an KCollectionListener which will be notified whenever elements are added to or removed from the KObservableCollection. If the same listener is added more than once, then it will be notified more than once.

Parameters

aListener - the KCollectionListener to be registered.
