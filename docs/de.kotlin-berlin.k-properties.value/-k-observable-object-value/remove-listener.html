removeListener

fun removeListener(aListener: KInvalidationListener): Unit

Removes the given listener from the list of listeners, that are notified whenever the value of the KObservable becomes invalid.

If it had been previously added then it will be removed. If it had been added more than once, then only the first occurrence will be removed.

Parameters

aListener - the KInvalidationListener to be removed.
