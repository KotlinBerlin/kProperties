addListener

open fun addListener(aListener: KMapListener<K, V>): Unit

Adds an KMapListener which will be notified whenever key value mappings are added to, removed from, or replaced in the KObservableMap. If the same listener is added more than once, then it will be notified more than once.

Parameters

aListener - the KListListener to be registered.
