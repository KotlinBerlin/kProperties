BasicKObservableMapValue()

Base class for all KObservableMapValue implementations that implements all the methods of an MutableMap and delegates them to the wrapped value. If the wrapped value is null then all methods behave as if applied to an immutable empty map. All methods of a mutable map throw an exception in this case.
