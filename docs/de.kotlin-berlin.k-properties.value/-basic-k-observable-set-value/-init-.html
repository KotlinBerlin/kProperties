BasicKObservableSetValue()

Base class for all KObservableSetValue implementations that implements all the methods of an MutableSet and delegates them to the wrapped value. If the wrapped value is null then all methods behave as if applied to an immutable empty set. All methods of a mutable set throw an exception in this case.
