KProperty

interface KProperty<T> : KWritableValue<T>, KReadOnlyProperty<T>

In addition to an KReadOnlyProperty a KProperty also implements the KWritableValue interface and so its value can be modified.

Additionally it offers support to bind it to another KProperty instance either uni- or bidirectional with the bind and bindBidirectional methods. To remove a binding there are the unbind and the unbindBidirectional methods.

The value of a KProperty that is bound unidirectional to another KProperty may not be changed manually.
