toBooleanBinding

fun <B : Boolean?> KObservableValue<B>.toBooleanBinding(): KBooleanBinding<B>

Returns a KBooleanBinding. If this already is an instance of KBooleanBinding it is simply returned. Otherwise a new KBooleanBinding is created which observes this KObservableValue for changes.
